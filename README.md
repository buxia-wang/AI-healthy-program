# AI健康助手 第一版

这是一个给亲朋好友试用的 H5/PWA 静态健康工具。它可以计算 BMI、BMR、TDEE，记录每日饮食热量和运动消耗，并根据特殊人群标签给出谨慎提醒。

## 使用方式

直接双击 `index.html` 即可打开。  
如果想体验 PWA 离线缓存，可以在本文件夹启动一个本地服务器后访问，例如：

```powershell
python -m http.server 5173
```

然后在浏览器打开：

```text
http://localhost:5173
```

## 当前包含

- 基础档案：性别、年龄、身高、体重、腰围、活动水平、目标
- 特殊情况：儿童青少年、老人、糖尿病、高血压、高血脂、生理期、孕期、健身、肾病/痛风、心血管病史
- 计算：BMI、BMR、TDEE、目标热量、腰高比提示
- 饮食记录：内置常见食物热量，也支持自定义热量
- 运动记录：用 MET 估算运动消耗
- 今日总览：摄入、运动消耗、相对目标差值
- 今日总览可编辑：可以直接填写今日总摄入和总运动消耗；留空时按清单自动汇总
- 健康概况分析：综合 BMI、腰高比、目标热量、当天热量记录、年龄和特殊人群标签生成判断依据与优先建议
- 高级简洁 UI：使用更接近一线产品官网和系统应用的浅色、细边界、强层级界面
- 固定顶部操作：保存后顶部只保留项目名和“修改数据”；点击后展开保存/清空，保存后自动收起
- 本地保存：数据保存在浏览器 `localStorage`

## 医疗边界

本工具仅用于健康科普、热量估算和生活方式记录，不提供医学诊断、治疗、处方或用药调整建议。儿童青少年、孕期或哺乳期、老人、糖尿病、高血压、高血脂、肾病、痛风、心血管病史、严重肥胖或近期身体不适者，请优先咨询医生、营养师或运动康复专业人士。
